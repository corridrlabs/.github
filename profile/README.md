# corridrlabs

[![Corridor banner](assets/corridor-banner.svg)](https://corridor-flax.vercel.app)

**Corridor, the developers' financial cloud:** open tooling for global payments, treasury, payroll rails, and programmable money workflows.

## Learn About Corridor 📚
- Read docs and integration guides in [docs](https://github.com/corridrlabs/docs).
- Explore docs website source in [docs-site](https://github.com/corridrlabs/docs-site).
- Use API contracts from [openapi](https://github.com/corridrlabs/openapi) and [api-specs](https://github.com/corridrlabs/api-specs).
- Build AI-native financial automations via [mcp](https://github.com/corridrlabs/mcp) and [mcp-tools](https://github.com/corridrlabs/mcp-tools).
- Start quickly with [examples](https://github.com/corridrlabs/examples) and [starters](https://github.com/corridrlabs/starters).
